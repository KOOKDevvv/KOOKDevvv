## KOOK

중학교 2학년이고, Flutter로 **바로장터**를 만들고 있습니다.
기능이 동작하는 데서 끝내지 않고 실제로 쓰기 편한 서비스가 되도록 계속 다듬고 있습니다.

|  |  |
|:--|:--|
| `NOW` | 바로장터 개발 |
| `FOCUS` | 상품 등록 · 채팅 · 결제 · 알림 |
| `STACK` | Flutter · Dart · Supabase · PostgreSQL |

---

### 바로장터

동네 사람끼리 물건을 사고팔고 이야기할 수 있는 장터·커뮤니티 앱입니다.

- 상품을 등록하고 발견하는 흐름
- 사용자 사이의 채팅과 거래 과정
- 결제, 알림, 프로필처럼 실제 운영에 필요한 기능
- 작은 화면에서도 자연스럽게 이어지는 모바일 UI

아직 개발 중이며, 새로운 기능보다 기존 흐름의 완성도를 높이는 작업에 집중하고 있습니다.
